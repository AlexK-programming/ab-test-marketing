## 📌 О проекте

В рамках подготовки портфолио для стажировки по анализу данных проведён **A/B-тест** для сравнения эффективности трёх рекламных промо (1, 2 и 3) в сети быстрого питания.

**Цель:** определить, какое промо приводит к наибольшим продажам.

**Данные:**  
- 548 наблюдений  
- 7 переменных: размер рынка (`MarketSize`), возраст магазина (`AgeOfStore`), продажи (`SalesInThousands`), номер промо (`Promotion`), неделя (`week`) и др.

---

## 📊 Основные этапы анализа

1. **EDA и проверка качества данных**  
   - Проверка пропусков, типов данных  
   - Описательные статистики (`describe()`)

2. **Визуальный анализ**  
   - Boxplot продаж по группам  
   - Оценка медиан, разброса, выбросов, асимметрии

3. **Проверка рандомизации**  
   - Сравнение групп по `MarketSize` и `AgeOfStore`  
   → группы сбалансированы

4. **Проверка предположений для ANOVA**  
   - Нормальность (тест Шапиро–Уилка) — нарушена, но ANOVA устойчива при больших выборках  
   - Равенство дисперсий (тест Ливена) — выполнено

5. **ANOVA**  
   - Выявлены статистически значимые различия между группами (F = 21.95, p < 0.001)

6. **Post-hoc анализ (Tukey HSD)**  
   - Попарное сравнение с доверительными интервалами

7. **Дополнительные метрики**  
   - Средние и медианные продажи  
   - Относительный эффект в %

## 🔍 Результаты

| Промо | Средние продажи (тыс. $) | Относительный эффект |
|-------|--------------------------|----------------------|
| 1     | 58.10                    | лучший               |
| 3     | 55.36                    | +16.98% vs Промо 2   |
| 2     | 47.33                    | худший               |

- **Промо 1** превосходит Промо 2 на **22.76%** (p < 0.001)  
- **Промо 3** превосходит Промо 2 на **16.97%** (p < 0.001)  
- Разница между Промо 1 и Промо 3 (**+4.95%**) статистически **не значима** (p = 0.244)

Внедрять **Промо 1** (как числовой лидер) или **Промо 3**. Промо 2 неэффективно.
